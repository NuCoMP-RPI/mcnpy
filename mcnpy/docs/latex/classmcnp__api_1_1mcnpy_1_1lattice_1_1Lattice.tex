Class for lattices. 

Defined by max indicies {\ttfamily i}, {\ttfamily j}, {\ttfamily k}, and a 3D array {\ttfamily lattice}. The array should be defined uning {\ttfamily numpy.\+array()} where {\ttfamily k} indicies are your outermost dimension followed by {\ttfamily j} and {\ttfamily i}. Elements of {\ttfamily lattice} can be {\ttfamily Universe} objects or simply their IDs. For the latter, {\ttfamily universes = dict()} where its keys are universe IDs and its values are the {\ttfamily Universe} objects.
