A full core model of RPI\textquotesingle{}s Reactor Critical Facility. 

A very critical reactor that will live on in the virtual world no matter what Shirley and the RPI administration does. This example will generate a new MCNP deck, build the \mbox{\hyperlink{classmcnp__api_1_1mcnpy_1_1example_1_1RCF}{RCF}}, and write the model to file.{\ttfamily water} sets the height of the water in the reactor in inches (default is {\ttfamily 68.\+0in}). {\ttfamily bank} sets the control rod bank height in inches (default is rods fully bottomed at {\ttfamily 0.\+0in}). {\ttfamily sporty} when set to {\ttfamily True} removes the center fuel pin which puts the \mbox{\hyperlink{classmcnp__api_1_1mcnpy_1_1example_1_1RCF}{RCF}} in sport mode. {\ttfamily filename} specifies the name of the new MCNP input (default is {\ttfamily ./mcnp\+\_\+inps/rcf\+\_\+full\+\_\+api.mcnp}). The model can be accessed with the {\ttfamily model} attribute.
