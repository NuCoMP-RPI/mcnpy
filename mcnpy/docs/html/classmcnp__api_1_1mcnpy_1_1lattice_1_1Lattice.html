Class for lattices.

Defined by max indicies i, j, k, and a 3D array lattice. The array should be defined uning numpy.array() where k indicies are your outermost dimension followed by j and i. Elements of lattice can be Universe objects or simply their IDs. For the latter, universes = dict() where its keys are universe IDs and its values are the Universe objects.
